# Test Cases – E-commerce Website

TC-01
Scenario: Search for a product

Steps:
1. Enter product name in search bar
2. Click search

Expected Result:
Relevant products should be displayed.

---

TC-02
Scenario: View product details

Steps:
1. Click a product from search results

Expected Result:
Product details page should display correct information.

---

TC-03
Scenario: Add product to cart

Steps:
1. Select a product
2. Click Add to Cart

Expected Result:
Product should be added to shopping cart.

---

TC-04
Scenario: Remove item from cart

Steps:
1. Go to cart
2. Click remove item

Expected Result:
Item should be removed from cart.
